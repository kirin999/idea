# README

# アイディア製造機

## 開発環境

### OS
* macOS Mojave バーション10.14.6

### 使用言語
* ruby 2.5.1p57
* Rails 5.2.4.3
* HTML5
* CSS3


### 依存ライブラリ
* Bundler version 2.1.4
* mysql  Ver 8.0.21 for osx10.14 on x86_64 (Homebrew)

### git clone後
ターミナルでbundle installを実行

```
bundle install
```

### 起動方法
railsサーバー立ち上げ

```
rails s
```
サーバー立ち上げ後localhost:3000にリクエスト
